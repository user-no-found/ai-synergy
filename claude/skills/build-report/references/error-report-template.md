# 编译错误报告模板

```markdown
## 编译错误报告

### 错误摘要
- 错误阶段：{任务名称}
- 错误类型：{编译错误/链接错误/依赖缺失/...}

### 错误详情
- 文件：{文件路径}
- 行号：{行号}
- 错误信息：
  ```
  {完整错误信息}
  ```

### 责任追溯
- 责任提案：{proposal_id}
- 责任子代理：{子代理名}
- 追溯依据：根据错误文件路径匹配提案中的错误追溯映射表

### 修复流程
1. 用户通知 `{责任子代理}` 修复问题
2. 责任子代理修复完成后通知用户
3. 用户重新启动 `build-agent` 执行编译
```

## 追加项目记录

编译失败后，追加记录到 `Record/record.md`：

```markdown
## YYYY-MM-DD HH:MM 编译结果
状态：失败
错误类型：{错误类型}
责任提案：{proposal_id}
错误摘要：{一句话描述}
```
